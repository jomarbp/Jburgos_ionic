SOLUCIONAR ERRORES DE NPM TEMPORALES:
==============================================
1. 
Linea::

  npm install --save --legacy-peer-deps

2. 
Linea::

  npm config set legacy-peer-deps true

3. 
Linea::

  npm cache clean --force
